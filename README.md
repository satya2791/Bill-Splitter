# Bill Splitter

A responsive Bill Splitter web application built using HTML, CSS, and JavaScript.

## Features

* Split bills equally among multiple people
* Calculate tip percentage automatically
* Responsive design for mobile, tablet, and desktop
* Simple and clean user interface

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Use

1. Enter the number of persons.
2. Enter the total bill amount.
3. Enter the tip percentage.
4. Click the **Start** button.
5. The application calculates how much each person should pay.
